## Rick And Morty Characters Sample App

iOS App (16.0+) with list, search and filter. Also a details screen.

## Api
This projects consumes [Rick and Morty Api](https://rickandmortyapi.com/).

## Requirements
- Xcode 16.0+

## Patterns

- SwiftUI
- MVVM
- Dependency Injection by constructor

## Test Coverage

- 88%
  - Network layer
  - Service layer
  - Model layer
  - ViewModel layer
- Views are not being tested

## Observability

- None, app is not on production, so this was not in the planning.
